# ShopSphere

ShopSphere is an Android product catalog application developed using Kotlin and MVVM architecture.

## Features
- Fetches products from DummyJSON API
- Search products by title
- Product listing with image, title, price, and rating
- Detailed product view
- Modern Material Design UI
- MVVM Architecture
- Retrofit API Integration
- RecyclerView for product display
- Coil for image loading

## Technologies Used
- Kotlin
- XML
- MVVM
- Retrofit
- LiveData
- ViewModel
- RecyclerView
- Coil
- Material Design

## Public API Used
https://dummyjson.com/products
